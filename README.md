## Scoring Strategy

### If Only Studying Tier A Topics:

- **Realistic Score:** 55-65 marks (out of 70)
- **Marks Distribution:**
  - Section A: 12-14 marks (most topics from Tier A)
  - Section B: 20-24 marks (choose CRC, subnetting, TCP/UDP)
  - Section C: 23-28 marks (choose routing, RSA, DNS)

### If Studying Tier A + Tier B Topics:

- **Realistic Score:** 65-70+ marks (out of 70)
- **Marks Distribution:**
  - Section A: 13-14 marks (complete coverage)
  - Section B: 24-30 marks (all choices available)
  - Section C: 28-35 marks (full flexibility)

### Exam Strategy to Score 60+:

1. **Section A (14 marks, 2 min per Q):** Answer all - focus on definitions from Tier A (9-10 marks)
2. **Section B (21 marks, choose 3 of 5 - 10 min per Q):**
   - Mandatory: CRC calculation, IP subnetting, TCP vs UDP
   - Optional: Cryptography, Congestion control
   - Expected: 18-21 marks if well-prepared
3. **Section C (35 marks, choose 1 of each 5 blocks - 15 min per Q):**
   - Always have 1 routing algo, 1 protocol, 1 diagram-based, 1 cryptography, 1 calculation ready
   - Expected: 20-25+ marks with diagrams

**Key to High Marks:** Add well-labeled diagrams, tables, and flow charts in every section C answer. This increases marks by 30-40% compared to prose-only answers.

## Study Plan: Day-Wise Order (7-10 days intensive)

### Days 1-2: Foundations (Unit 1 - Tier A only)

- OSI Model (7 layers, functions, comparison with TCP/IP)
- Network topologies (mesh, star, ring, bus - pros/cons with diagrams)

### Days 3-4: Link Layer Mastery (Unit 2 - Tier A focus)

- CRC algorithm (min 5 worked examples, understand remainder division)
- Error correction (Hamming code calculation)
- STOP N WAIT, Sliding window protocol (efficiency formula)

### Days 5-6: Network Layer (Unit 3 - Tier A focus)

- Subnetting (Class A/B/C, CIDR, variable length subnetting - 10 examples)
- Routing algorithms (DVR, count-to-infinity, LSR basic concept)

### Days 7: Transport Layer (Unit 4 - Tier A focus)

- TCP vs UDP (7-point comparison table, header comparison)
- Connection management (3-way handshake with diagram)

### Days 8-9: Application Layer (Unit 5 - Tier A focus)

- DNS, FTP, SNMP (what each does, how they work)
- RSA algorithm (key generation, encryption/decryption with numbers)

### Days 10+: Tier B Topics (optional buffer)

- TCP congestion control, Cryptography basics, Token bucket, IPv6
